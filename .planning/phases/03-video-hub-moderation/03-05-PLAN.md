- For each report: fetch the flagged video from `videos` collection
- Group reports by videoId (multiple users may flag the same video) — aggregate flag reasons and count
- Render FlaggedContentCard for each unique flagged video
- Empty state: "No Flagged Content" / "No content has been reported. The community is in good order."

- Handles channel approve/reject: calls approveChannel/rejectChannel Server Action
- Shows loading states per-card during action processing
