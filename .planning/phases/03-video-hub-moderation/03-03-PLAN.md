**2. Create `src/components/video/VideoUploadForm.tsx`** as a `'use client'` component:

Props: `{ uid: string; channels: { channelId: string; name: string; handle: string }[] }`

State variables:
- `progress` (number, 0-100)
- `phase` ('idle' | 'uploading' | 'processing' | 'done' | 'error')
- `errorMessage` (string | null)

Form fields (all inside a Card component):
- **Video file**: `<input type="file" accept="video/mp4,video/webm,video/quicktime" />` — validate max 2GB client-side before upload
- **Thumbnail**: `<input type="file" accept="image/*" />` — optional
- **Title**: Input component, required, maxLength 200
- **Description**: `<textarea>` with `font-garamond text-base bg-navy-mid border border-gold/[0.15] rounded-[6px] p-3`, maxLength 5000
- **Tags**: Input component with comma-separated entry, max 10 tags
- **Category**: native `<select>` with ORTHODOX_CATEGORIES options, `bg-navy-mid border border-gold/[0.15] rounded-[6px] p-2 text-text-light font-garamond`
- **Channel**: native `<select>` — "No channel (profile only)" as default, then list user's approved channels; optional
- **Submit**: Button gold variant "Upload Video", disabled during upload

**Duration detection:** After user selects video file, create a temporary `URL.createObjectURL(file)` and a hidden `<video>` element. Listen for `loadedmetadata` event, read `videoElement.duration`, store as `durationSeconds`. Revoke object URL after.

**Upload flow on submit:**
1. Generate videoId via `crypto.randomUUID()`
2. If thumbnail file exists, upload to `thumbnails/${videoId}/${thumbFile.name}` via `uploadBytesResumable`, get thumbnailUrl
3. Upload video file to `videos/${uid}/${videoId}/${file.name}` via `uploadBytesResumable`
4. On `state_changed`: update progress state
5. On error: set phase='error', errorMessage='Upload failed. Check your connection and try again. Large files may need a stable connection.'
6. On complete: set phase='processing', call `getDownloadURL(uploadTask.snapshot.ref)`, then call `createVideo(uid, { videoId, videoUrl, storagePath, title, description, tags, category, thumbnailUrl, durationSeconds, channelId })` Server Action
7. On Server Action success: set phase='done', show "Pending Review" status badge, redirect to profile or channel page after 2s
8. On Server Action error: set phase='error', show error message

Import firebase client SDK: `getStorage, ref, uploadBytesResumable, getDownloadURL` from `firebase/storage` and `firebaseApp` from `@/lib/firebase/client`.
