Create `src/app/actions/videos.ts` with `'use server'` directive at top. Import from `@/lib/firebase/admin` (getAdminFirestore), `firebase-admin/firestore` (FieldValue), `next-firebase-auth-edge` (getTokens), `next/headers` (cookies), `@/lib/auth/config` (authConfig), `@/lib/firestore/videos` (buildVideoSearchKeywords, getVideoById), `@/lib/firestore/profiles` (getProfileByUid), `@/lib/constants/categories` (ORTHODOX_CATEGORIES), and `zod` for validation.

**createVideo(uid, data):**
- Validate with Zod: title (1-200 chars), description (0-5000), tags (string array, max 10), category (must be in ORTHODOX_CATEGORIES), videoUrl (string URL), storagePath (string), thumbnailUrl (string | null), durationSeconds (number >= 0), channelId (string | null)
- Fetch uploader profile via getProfileByUid(uid); return error if not found
- Fetch uploader role from getTokens cookie check (roleLevel)
- Build searchKeywords via buildVideoSearchKeywords(title, description, tags)
- If channelId is provided, fetch channel via getChannelById to get channelHandle
- Write to `videos/{data.videoId}` with all Video fields, status always `'pending_review'`, viewCount/likeCount/commentCount all 0, moderator fields all null
- Return `{ success: true }` or `{ success: false; error: string }`

**updateVideoStatus(videoId, decision, note?):**
- Get caller tokens from cookies; require roleLevel >= 2; return `{ error: 'Insufficient permissions.' }` if not
- Validate decision is one of: `'published' | 'rejected' | 'changes_requested'`
- If decision is `'changes_requested'`, require note to be non-empty string
- Update `videos/{videoId}` with: status=decision, moderatorNote=note??null, moderatedAt=FieldValue.serverTimestamp(), moderatedBy=caller uid
- Read video doc to get uploaderUid
- Write notification to `users/{uploaderUid}/notifications/{auto-id}` with:
  - notificationId: auto-generated doc ID
  - type: 'moderation'
  - fromUid: caller uid (moderator)
  - fromHandle: '' (moderator identity not disclosed)
  - fromDisplayName: 'Moderator'
  - fromAvatarUrl: null
  - postId: null
  - postText: null
  - videoId: videoId
  - decision: decision
  - moderatorNote: note ?? null
  - read: false
  - createdAt: FieldValue.serverTimestamp()
- Return `{ success: true }`

**deleteVideo(videoId, callerUid):**
- Fetch video; verify callerUid === video.uploaderUid OR roleLevel >= 2
- Batch delete: video doc, all likes subcollection docs, all comments subcollection docs
- Use 500-op batch chunks (same pattern as deletePost)
- Return `{ success: true }`

**likeVideo(videoId, uid):**
- Check if `videos/{videoId}/likes/{uid}` exists
- If exists: delete it, decrement likeCount with FieldValue.increment(-1)
- If not exists: create it with { uid, createdAt: FieldValue.serverTimestamp() }, increment likeCount with FieldValue.increment(1)
- Write like notification to video uploader (read video doc for uploaderUid); skip if liker is uploader
- Return `{ liked: boolean }`

**incrementViewCount(videoId):**
- Update `videos/{videoId}` with `viewCount: FieldValue.increment(1)`
- No auth check needed (view counts increment on page load)
- Return `{ success: true }`

**1. Create `src/app/actions/channels.ts`** with `'use server'` directive.

**createChannelApplication(uid, data):**
- Validate with Zod: name (1-100), handle (3-50, lowercase alphanumeric + hyphens, regex: `/^[a-z0-9][a-z0-9-]*[a-z0-9]$/`), channelType ('personal' | 'institutional'), description (0-2000), primaryCategory (must be in ORTHODOX_CATEGORIES)
- Check handle uniqueness via isChannelHandleAvailable(handle); return error if taken
- Generate channelId via crypto.randomUUID() (import from 'crypto' in Node)
- Write to `channels/{channelId}` with: all fields, ownerUid=uid, logoUrl=null, bannerUrl=null, subscriberCount=0, videoCount=0, status='pending_approval', createdAt=FieldValue.serverTimestamp(), approvedAt=null, approvedBy=null
- Return `{ success: true, channelId }`

**approveChannel(channelId):**
- Get caller tokens; require roleLevel >= 2
- Update `channels/{channelId}` with: status='approved', approvedAt=FieldValue.serverTimestamp(), approvedBy=caller uid
- Write notification to channel owner: type='moderation', decision='published' (reuse moderation notification pattern), videoId=null, moderatorNote='Your channel application has been approved.'
- Return `{ success: true }`

**rejectChannel(channelId, reason):**
- Get caller tokens; require roleLevel >= 2
- Update `channels/{channelId}` with: status='rejected'
- Write notification to channel owner: type='moderation', decision='rejected', moderatorNote=reason
- Return `{ success: true }`

**subscribeChannel(uid, channelId):**
- Check if `channelSubscribes/{channelId}/subscribers/{uid}` exists; return early if already subscribed
- Write subscriber doc: `{ uid, createdAt: FieldValue.serverTimestamp() }`
- Increment `channels/{channelId}.subscriberCount` with FieldValue.increment(1)
- Return `{ subscribed: true }`

**unsubscribeChannel(uid, channelId):**
- Check if subscriber doc exists; return early if not subscribed
- Delete subscriber doc
- Decrement `channels/{channelId}.subscriberCount` with FieldValue.increment(-1)
- Return `{ subscribed: false }`

**createVideoComment(uid, videoId, text):**
- Get caller profile via getProfileByUid(uid); return error if not found
- Validate text (1-2000 chars) with Zod
- Generate commentId
- Write to `videos/{videoId}/comments/{commentId}` with: all VideoComment fields, isEdited=false, createdAt/updatedAt
- Increment `videos/{videoId}.commentCount` with FieldValue.increment(1)
- Write comment notification to video uploader (read video doc for uploaderUid); skip if commenter is uploader
- Return `{ success: true, commentId }`

**deleteVideoComment(uid, videoId, commentId):**
- Read comment doc; verify uid === comment.authorUid OR roleLevel >= 2
- Delete comment doc
- Decrement `videos/{videoId}.commentCount` with FieldValue.increment(-1)
- Return `{ success: true }`

**editVideoComment(uid, videoId, commentId, newText):**
- Read comment doc; verify uid === comment.authorUid
- Validate newText (1-2000 chars)
- Update comment doc with: text=newText, isEdited=true, updatedAt=FieldValue.serverTimestamp()
- Return `{ success: true }`
