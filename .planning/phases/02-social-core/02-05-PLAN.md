**1. Create `src/app/(main)/agora/[postId]/page.tsx`** — Server Component:

- Auth check: `getTokens(await cookies(), authConfig)`, redirect to `/login` if no tokens
- Extract `postId` from params: `const { postId } = await params;`
- Call `getPost(postId)` from `@/lib/firestore/posts`
- If post not found, call `notFound()`
- Extract `uid` from `tokens.decodedToken.uid`
- Check if user follows the post author (for follower-only comment restriction display): query `follows/{uid}_{post.authorUid}` existence
- Render the post using the `PostCard` component at the top (reuse from plan 02-04)
- Render `PostDetailClient` below, passing `postId`, `currentUserUid: uid`, `commentsRestricted: post.commentsRestricted`, `isFollowing`, `isPostAuthor: uid === post.authorUid`

**2. Create `src/components/agora/PostDetailClient.tsx`** — Client Component:

Props: `postId: string`, `currentUserUid: string`, `commentsRestricted: 'all' | 'followers'`, `isFollowing: boolean`, `isPostAuthor: boolean`

- On mount: load ALL comments from `posts/{postId}/comments` subcollection ordered by `createdAt asc` using Firestore client SDK `getDocs` (per CONTEXT: all comments load at once, no pagination)
- State: `comments: Comment[]`, `loading: boolean`, `commentText: string`
- **Comment compose area**:
  - If user is registered AND (commentsRestricted === 'all' OR isFollowing OR isPostAuthor): show textarea
  - Textarea: styled like Input, `min-height 60px`, placeholder "Add a comment..."
  - Submit button: `Button` gold variant, text "Comment", disabled when empty
  - On submit: optimistic add (clear textarea, append comment to list), call `createComment(currentUserUid, postId, commentText)`. On failure: remove optimistic comment, repopulate textarea, show toast "Your comment could not be posted. Please try again."
  - If `commentsRestricted === 'followers'` and NOT isFollowing and NOT isPostAuthor: show message "Only followers can comment on this post." (per UI-SPEC copy)
  - If user is guest (no currentUserUid — handled by auth redirect, but defensive): show "Sign in to join the conversation." with link to `/login`
- **Comment list**: flat list of `CommentCard` components, ordered by createdAt asc
- Show total count: `{comments.length} Comments` heading above list

**3. Create `src/components/agora/CommentCard.tsx`** — Client Component:

Props: `comment: Comment`, `currentUserUid: string`, `postId: string`, `onCommentDeleted?: (commentId: string) => void`, `onCommentEdited?: (commentId: string, newText: string) => void`

Per UI-SPEC CommentCard:
- Simpler than PostCard — no image, no category chip, no jurisdiction badge
- **Header row**: Avatar (28px circle) + `@handle` (`font-cinzel text-xs text-text-mid`) + relative timestamp (`font-cinzel text-xs text-text-mid`)
- "edited" marker: if `comment.isEdited`, show `font-garamond text-xs italic text-text-mid` text "edited" after timestamp
- **Comment text**: `font-garamond text-base text-text-light`
- **Actions** (only if `comment.authorUid === currentUserUid`):
  - Edit button: `text-text-mid text-xs hover:text-text-light` — on click, switch to inline edit mode (replace text with textarea, show Save/Cancel buttons)
  - Delete button: `text-text-mid text-xs hover:text-crimson` — on click, confirmation: "Delete this comment? This cannot be undone." Confirm: crimson "Delete", cancel: outline "Keep"
  - On edit save: call `editComment(currentUserUid, postId, comment.commentId, newText)`, call `onCommentEdited` callback
  - On delete confirm: call `deleteComment(currentUserUid, postId, comment.commentId)`, call `onCommentDeleted` callback
