## Spacing Scale

Declared values (must be multiples of 4). Source: Tailwind v4 default scale + codebase pattern scan.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding (`px-2 py-0.5`) |
| sm | 8px | Compact element spacing, card internal gaps (`gap-1`) |
| md | 16px | Default element spacing, card padding (`p-4`), page horizontal padding (`px-4`) |
| lg | 24px | Section padding, vertical rhythm between content blocks |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks (e.g. gap between Fathers grid and search bar) |
| 3xl | 64px | Page-level top/bottom padding (`py-8` = 32px body + 70px nav offset) |

Exceptions:
- Navigation bar height: 70px (`--spacing-nav: 70px`) — pre-existing token, do not change
- Author card avatar: 48px (w-12 h-12) — matches MemberCard pattern
- Touch targets: minimum 44px height on all interactive elements (WCAG 2.5.5)

## Typography

All sizes extracted from existing codebase patterns. Source: Scripture landing page, Navbar, MemberCard, Button component.

| Role | Font Family | Size | Weight | Line Height | Usage |
|------|-------------|------|--------|-------------|-------|
| Display | Cinzel (`font-cinzel`) | 20px (`text-xl`) | 400 | 1.2 | Page h1 titles — "Church Fathers", "Study Guides", author name on author page |
| Heading | Cinzel (`font-cinzel`) | 12px (`text-xs`) uppercase + `tracking-widest` | 400 | 1.2 | Section labels, era dividers, card metadata labels |
| Body | EB Garamond (`font-garamond`) | 16px (`text-base`) | 400 | 1.5 | Patristic text reading view body, author bio, study guide item descriptions |
| Label | Cinzel (`font-cinzel`) | 14px (`text-sm`) | 400 | 1.4 | Author card name, study guide titles in list, search result titles |

Notes:
- Body text in the reading view uses `font-garamond` at `text-base` (16px) with `leading-relaxed` (1.625) — matches the Scripture reader's verse rendering approach
- Cinzel Decorative (`font-cinzel-dec`) is reserved for the site logo only — do not use for phase-level headings
- Two weights in use: 400 (regular) for all body and label copy; no semibold or bold weight is used anywhere in the existing codebase. Match this: **400 only**.

## Color

Source: `src/styles/globals.css` — @theme block. All values are pre-existing CSS custom properties.

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `bg-navy` | `#0d1b2e` | Page background, primary surface |
| Secondary (30%) | `bg-navy-mid` | `#152338` | Author cards, study guide cards, search result list background, sidebar, navbar |
| Tertiary surface | `bg-navy-light` | `#1e3352` | Hover states on cards, active/selected states |
| Accent (10%) | `text-gold` / `border-gold` | `#c9a84c` | See reserved list below |
| Muted text | `text-text-light` | `#e8dfc8` | Body text, card titles |
| Dim text | `text-text-mid` | `#b8a888` | Metadata, attribution, era labels, secondary copy |
| Destructive | `text-crimson` | `#8b1a1a` | Sign Out only — no destructive actions exist in this phase |

Accent (`#c9a84c`) is reserved ONLY for:
- Page h1 text (`text-gold`)
- Section label text (`text-gold`)
- Card author name text (`text-gold`)
- Active search result keyword highlight
- Card border hover state (`hover:border-gold/40`)
- Navigation links hover state (`hover:text-gold`)
- "Church Fathers" navbar link text (active state)
- Study guide step number badges
- "Previous / Next" chapter navigation links in the patristic text reader

Do NOT use gold on: body copy, era badges, topic filter tabs (inactive), metadata timestamps, attribution footnotes.

Border treatment (established pattern): `border-gold/[0.15]` at rest, `hover:border-gold/40` on hover. Applied to all card and input borders.

## Page Layout Contracts

### `/fathers` — Church Fathers Landing Page

```
max-w-7xl mx-auto px-4 py-8
├── h1: "Church Fathers" — font-cinzel text-gold text-xl uppercase tracking-widest mb-6
├── PatristicSearch (search input, full width, mb-8)
└── Two-column layout (lg:grid lg:grid-cols-[240px_1fr] gap-8)
```

## Accessibility Contract

- All interactive cards must be focusable via keyboard (`tabIndex={0}` or use `<a>` / `<Link>`)
- Era badge text must meet 4.5:1 contrast against `bg-navy-mid` background
- Topic filter pills must have visible focus ring (`focus-visible:ring-2 focus-visible:ring-gold/60`)
- Search input requires `<label>` or `aria-label="Search Church Fathers texts"`
- Study guide step numbers must not be the only indicator of order — screen reader must read step number inline with title
- Patristic text reader uses `<article>` semantic element wrapping the body text
- Author page name h1 must be unique per page (not duplicated in breadcrumb)
